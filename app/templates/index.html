Welcome to the German Gender API

With it, you can get the grammatical gender of German nouns in a very simple and practical way.

How about accessing german-gender-api.deta.dev/api/v1/nouns/Mädchen instead of searching "what's the gender of Mädchen in German"?

If it's helpful to you, you can simply search for the nouns you want. If you are a developer looking for an API like this to use in your website, there's also a pageable object of all nouns in the database.

For further details, access the Documentation.
